# jeniussystems.ai

Source for the **jenius Systems** one-page site — a static landing page describing
a personal AI assistant project. Hosted free on GitHub Pages.

- Single static page. No backend, no tracking, no build step required to serve.
- `styles.css` is a precompiled Tailwind stylesheet (no runtime CDN).
  To rebuild after editing `index.html`:
  `npx tailwindcss@3 -c tailwind.config.js -i tailwind.input.css -o styles.css --minify`

This is a personal project. Nothing here is for sale.
